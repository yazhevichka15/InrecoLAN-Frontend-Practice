# Практика по Frontend-разработке (ИнрэкоЛАН)

Разработка фронтенд-части веб-приложения "Империя люстр"

## Структура проекта (дорабатывается)

Используется архитектура Feature-Sliced Design

```
/
├── node-modules/
├── src/
│   ├── app/
│   │   ├── App.tsx
│   │   ├── index.tsx
│   │   └── index.css
│   ├── processes/
│   ├── pages/
│   ├── widgets/
│   ├── features/
│   ├── entities/
│   └── shared/
├── index.html
├── package.json
├── package-lock.json
├── eslint.config.js
├── .prettierrc.json
├── tsconfig.json
├── tsconfig.app.json
├── tsconfig.node.json
├── vite.config.json
└── README.md
```

## Стек технологий

Tailwind, TypeSript, React, Redux
